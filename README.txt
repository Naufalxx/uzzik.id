Taruh foto-foto produk kamu di folder ini.

Setelah menaruh foto, buka file ../../data/products.js dan ubah
field "img" / "img2" produk terkait supaya menunjuk ke file ini,
misalnya:

  img: "images/products/chiron-1.jpg"

Rekomendasi:
- Format: JPG atau PNG
- Rasio gambar: mendekati 3:2 (misal 900x600 px)
- Ukuran file: usahakan di bawah 500KB per foto biar website tetap ringan
